# Feeding Program

## Project Summary

Feeding Program is a web-based system built to support the management of student feeding-related records and activities. The project uses a ReactJS frontend and a Python backend. It is designed to help users access the site locally, manage the database, and run the application with a simple setup process.

---

## Program Setup

To install the required dependencies for both the frontend and backend, simply run the setup file once:

1. Locate `setup.bat` in the project root folder..
2. Double-click `setup.bat`.
3. Wait for the setup process to finish.

You only need to do this once during the initial setup.

## Running the Site

To start the application:

1. Locate `start.bat` in the project root folder.
2. Double-click `start.bat`.
3. Wait for the site to open automatically.

If the browser does not open, you can access the site manually by going to:

`http://localhost:3000/login`
